# Website Analytics & Validation Dashboard

## Overview

This Power BI project analyzes website logs and validation data to help monitor application performance, identify critical errors, and track system health in real time.

## Features

- Executive Overview Dashboard
- Validation Dashboard
- Live Monitoring Dashboard
- Developer Dashboard
- Error Resolution Dashboard
- Error Details Report

## Key KPIs

- Total Logs
- Total Errors
- Warning Rate
- Error Rate
- Success Rate
- Failed Validation
- SMS Success Rate
- Most Affected Branch
- Latest Error
- Health Score

## Visualizations

- Error Trend
- Severity Distribution
- Error Categories
- Top Error Files
- Error by Module
- Weekly Error Trend
- Live Error Monitoring
- Monthly Error Analysis
- Error Resolution Status

## Tools Used

- Power BI
- Power Query
- DAX
- Excel
- Log File Analysis
